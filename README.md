# 🪨📄✂️ Piedra, Papel o Tijera

¡Bienvenido a mi proyecto para la materia de Programación 3! 

Decidí darle una vuelta de tuerca al clásico juego de Piedra, Papel o Tijera. En vez de jugar una sola vez y ya, armé un mini-torneo donde el primero que llegue a 3 puntos se lleva la corona. Además, le metí un poquito de suspenso y mensajes bastante directos para hacerlo más divertido.

## 🎮 ¿Cómo se juega?
1. Escribes tu nombre para entrar a la partida.
2. Eliges tu jugada usando los botones (Piedra, Papel o Tijera).
3. Aguantas el suspenso de 1.5 segundos mientras la computadora decide.
4. El primero que llegue a **3 puntos** es el campeón absoluto. Si pierdes... bueno, te tocará ponerte las pilas para la próxima.

## 🚀 ¿Cómo ponerlo a funcionar?
Cero estrés, no tienes que instalar programas raros ni librerías pesadas.
1. Descarga el archivo `index.html` (o clona este repositorio).
2. Dale doble clic al archivo `index.html`.
3. Se abrirá automáticamente en tu navegador web (Chrome, Edge, Firefox, el que uses) y ya estarás listo para jugar.

## 🛠️ Tecnologías que utilicé
Todo el proyecto está hecho puramente con las bases del desarrollo web:
* **HTML5:** Para armar la estructura y los botones.
* **CSS3:** Para el diseño, el fondo degradado oscuro y las animaciones de suspenso (los latidos del texto).
* **JavaScript (Vanilla):** Para toda la lógica pesada: generar la jugada aleatoria de la PC (`Math.random()`), llevar el marcador, manejar los tiempos con `setTimeout` y declarar al ganador.
